Frustration

This simple class calculates the chance of winning Frustration Solitaire given two values, Cards (13) and Suits (4). It's developed in PHP and uses Rook Polynomials to calculate the chance of winning.

BUG: Setting the number of cards and suits to larger numbers may return NaN

Can be instantiated like so: $frustration = new Likel\Frustration();

An example calculation for a normal deck of cards $number_of_cards = 13; $number_of_suits = 4; $frustration = new Likel\Frustration($number_of_cards, $number_of_suits); $chance = $frustration->getChance();
